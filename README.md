# A 10-Year Comparative Study on the Performance of Sensex, Gold, and HDFC Flexi Cap Fund (2015–2024)

## Project Overview

This MBA Finance Major Project evaluates and compares the long-term performance of three popular investment options in India:

- BSE Sensex
- Gold
- HDFC Flexi Cap Fund

The analysis covers the period from **2015 to 2024** and focuses on returns, risk, and investment performance using financial analysis techniques.

---

## Objectives

- Compare the performance of Sensex, Gold, and HDFC Flexi Cap Fund.
- Evaluate returns and risk over a 10-year period.
- Identify the most suitable investment based on different financial metrics.
- Present insights using Excel and Power BI dashboards.

---

## Tools Used

- Microsoft Excel
- Power BI
- Financial Ratio Analysis
- Statistical Analysis

---

## Key Metrics Analyzed

- CAGR (Compound Annual Growth Rate)
- Annual Returns
- Rolling Returns (3-Year & 5-Year)
- Standard Deviation
- Sharpe Ratio
- Beta
- Maximum Drawdown
- Risk-Return Analysis

## Key Findings

- HDFC Flexi Cap Fund generated the highest long-term CAGR.
- Sensex provided consistent long-term growth with moderate volatility.
- Gold acted as a defensive asset during market downturns.
- Risk-adjusted performance was evaluated using the Sharpe Ratio and Beta.
